# RTL Design and Verification of a UART-Based Luggage Sorting System Using Verilog HDL and SystemVerilog
# Project Overview
This project presents the RTL design and functional verification of a UART-based Smart Luggage Sorting System developed using Verilog HDL and SystemVerilog.
The system is designed to simulate an automated airport luggage classification and routing mechanism based on weight measurement. 
It integrates digital weight sensing logic, UART-based data communication, classification algorithms, and FSM-controlled conveyor routing into a unified VLSI design.
The primary objective of this project is to design a fully synthesizable hardware system that can classify luggage into different categories (Light, Medium, Heavy) 
and route them to appropriate conveyor belts using finite state machine control logic, while communicating weight data using UART protocol.
